# Frontend React ERP Template

## File Structure

```
.claude/

├── agents/
│   ├── erp-frontend-architect.md
│   └── react-frontend-architect.md
├── settings.local.json
└── skills/
    └── react-frontend/
        ├── SKILL.md
        └── references/
            ├── architecture.md
            ├── design-system.md
            ├── generation.md
            └── validation-guide.md
--

PRD/
    ├── Frontend-PRD.txt                    # BoundedContext level requirements
    └── Wireframe/
        └── wireframe.md                    # Component wireframe specifications
```

## Usage

### 1. Open Terminal
Open a terminal in your working directory.

### 2. Set Node Version (Node 20+)
```bash
nvm use 22
```

### 3. Run Claude Code
Execute Claude Code in your IDE.

### 4. Add PRD File to Prompt
Add `@Frontend-PRD.txt` to the prompt area and execute.
